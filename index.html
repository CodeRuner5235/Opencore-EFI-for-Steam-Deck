Opencore for Steam Deck

This project is for using Opencore to run MacOS on the Valve Steam Deck.

The MacOS versions that I will be targeting are Catalina (10.15) and Big Sur (11). The Project can be found here.

My reasoning here is that Apple changed a lot of things between Big Sur and Monterey so I will be focusing on these two versions for now. I will deninetly try to get Montery and up running in the future.
